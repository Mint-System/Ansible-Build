# Scripts

The following Ansible roles contain command line tools that can be used independent of Ansible.

## [ansible_scripts](roles/ansible_scripts/README.md)

```bash
curl -L https://ansible.build/ansible_scripts/install | bash
```

- ansible-vault-get
